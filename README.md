# Smart-University-Opening-Weather-Based-Notification-System
A C++ console app that automates university campus status decisions based on weather conditions using OOP principles. outputs ✅ Open | ⚠️ Caution | 🔴 Closed based on temperature, rain &amp; storm levels.
Key Highlights:

▪️ Applied core OOP principles — Encapsulation, Abstraction & Modularity

▪️ Designed two classes: Weather & UniversitySystem

▪️ Features: add, search & display weather records with real-time status check

▪️ Built using arrays, loops, conditionals & switch-case in C++
A great hands-on experience turning programming concepts into a real-world solution! 💡
